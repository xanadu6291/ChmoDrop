# ChmoDrop

ChmoDrop は macOS 向けのシンプルな GUI アプリで、ドロップされたフォルダをロック（書き込み禁止）にしたり、ロック解除したりするツールです。  
MIT ライセンスのもと、オープンソースとして公開しています。

## 🔧 機能

- ドロップされたフォルダをロックしたり、ロック解除したりする。
- ロック状態 / ロック解除状態を示すアイコンをフォルダに付ける。
- 公証済みアプリ（macOS の Gatekeeper 対応）

## 📦 ビルド方法

1. [Xcode](https://developer.apple.com/xcode/) をインストール
2. 本リポジトリをクローン：
   ```bash
   git clone https://github.com/xanadu6291/ChmoDrop.git
   cd ChmoDrop```
3. ChmoDrop.xcodeproj を開く
4.	Cmd + R でビルド＆実行！
